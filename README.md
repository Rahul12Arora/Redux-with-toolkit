# Redux-with-toolkit

**Get Started**

```
//Make a new project using redux

npx create-react-app my-app --template redux

--------------------------------------------
//Add to existing project

npm install redux
npm install react-redux (For React bindings)

--------
//To add redux toolkit
npm install @reduxjs/toolkit
```
